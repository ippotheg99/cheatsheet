Our first topic to understand will be :

The Box Model

'The Box Model' in CSS helps with the layout of a webpage. It is essentially a box that wraps around HTML elements and aids in defining space between elements.

Here are the parts of the model explained :

Aspect of model Examples Description
Content Titles, paragraphs, images etc. The content of the box, where text and images appear
Padding padding: 10px 10px, 20px, 20px Creates extra space within an element. With four values, it goes clockwise from top, right, bottom to left
Border border: 1px dotted gray A border that goes around the padding and content. A 1px dotted border that is gray
Margin margin: 10px 10px Margins create extra space around an element
